# 📱 PDM - Programação para Dispositivos Móveis

Repositório de atividades da disciplina de PDM — React Native com Expo (TypeScript).

## 🗂️ Branches

| Branch | Conteúdo | Status |
|--------|----------|--------|
| `main` | Projeto base | ✅ |
| `atividade-01` | Interpolação - Catálogo Dinâmico | ⬜ |
| `atividade-02` | Screen Wrappers | ⬜ |
| `atividade-03` | useState - Identificação de Visitante | ⬜ |
| `atividade-04` | useEffect - Sensor de Estacionamento | ⬜ |
| `pratica-01` | Prática de Implementação 1 - Formulário | ⬜ |

## 🚀 Como executar

```bash
# Instalar dependências
yarn install

# Iniciar o projeto
yarn start

# Ou diretamente para web
yarn web
```

## 📦 Tecnologias

- React Native + Expo
- TypeScript
- Expo Go (testes no celular)

## 📁 Estrutura do Projeto

```
src/
├── assets/              # Imagens e recursos estáticos
├── components/
│   ├── screen-wrappers/ # Wrappers reutilizáveis de tela
│   └── ui/              # Componentes de UI genéricos
├── types/               # Tipos TypeScript compartilhados
└── utils/               # Funções utilitárias
```
